# cash_flow_management

До запуска проекта в файле .env (в корне проекта) необходимо заполнить все переменные:
- первый блок - данные для подключения к базе данных;
- второй блок - данные для суперпользователя админ-панели Django.

Команда для запуска проекта (находясь в корне проекта): docker compose up --build.

После запуска создать с соответствующими учетными данными суперпользователя:
1. Зайти внутрь docker-контейнера с django app.
2. Выполнить команду python manage.py createsuperuser и дальнейшие инструкции в консоли (указать имя пользователя, почту и пароль).
